# Flex Checkout Flutter SDK

## Installation

Add to your `pubspec.yaml`:

```yaml
dependencies:
  flex_checkout_flutter:
    git:
      url: https://github.com/flexapp/flex-checkout-flutter.git
      ref: 1.2.2
```

## Documentation

See the [Flex Checkout Flutter docs](https://checkout.int.getflex.com/v2/rent/docs/flutter/configuration/installation) for full setup and usage.
